# Making Your Own Mini-XLR Cable

When I was building this project, HiFiBerry did not have cables in stock so I had to make my own. Here is what you need.

## Materials

- [Mini-XLR 4pin plugs](https://a.co/d/05eEJEnN)
- [Canare L-4E5C Mini-Star-Quad Microphone Cable by the Foot](https://www.markertek.com/product/l-4e5c-bk/canare-l-4e5c-mini-star-quad-microphone-cable-by-the-foot-black)

## Plug Disassembly

![Mini-XLR plug disassembled](images/IMG_4061.png)

1. Strip back the insulation, shielding and wires to appropriate length
2. Install the plug cover, plastic cover, and metal wire relief. **IMPORTANT. Do this before you solder both ends of the cable**
3. Tin the plug and wires with solder
4. Solder wires. Make sure wires are connected to same pin on the plug on each side
5. Screw on plug cover
